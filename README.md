# ECG Monitoring LoRaWAN

## Overview
The ECG Monitoring LoRaWAN project is designed to monitor and transmit electrocardiogram (ECG) data over a LoRaWAN network. This project aims to provide a reliable and efficient way to collect and analyze ECG data remotely, making it suitable for healthcare applications.

## Features
- ECG data monitoring
- Low-power consumption using LoRaWAN technology
- User-friendly web interface for data visualization
- Data storage and retrieval for historical analysis
- Support for multiple patients
